Dasturlash tillari

HTML

HTML – bu veb-sahifalarni yaratish uchun ishlatiladigan til. U saytning skeleti bo‘lib, unda nima bo‘lishini aytadi: matn, rasm, tugma yoki link. HTML’da teglar ishlatiladi. Teglar saytga nimani ko‘rsatishini bildiradi: sarlavha, paragraf, rasm yoki havola. Hujjat uch qismdan iborat: Birinchi qism sayt HTML ekanini bildiradi. Ikkinchi qism yashirin bo‘lib, unda sarlavha va boshqa ma’lumotlar joylashadi. Uchinchi qism esa foydalanuvchi ko‘radigan hamma narsani o‘z ichiga oladi. HTML faqat tuzilma yaratadi. Rang-barang qilish uchun CSS, harakat va interaktivlik qo‘shish uchun JavaScript ishlatiladi. Shunday qilib, HTML – bu saytni qurish uchun asosiy vosita, u saytning skeletini beradi va boshqa narsalarni unga qo‘shish mumkin bo‘ladi.

CSS

CSS – bu saytga rang, shakl va dizayn beradigan til. Agar HTML saytning skeleti bo‘lsa, CSS – uning kiyimi va bezagi hisoblanadi. CSS yordamida siz quyidagilarni qilishingiz mumkin: Matn rangini va o‘lchamini o‘zgartirish Fon rangini qo‘yish yoki rasm qo‘yish Tugmalarni, qutilarni chiroyli qilish Rasm va matnni joylashtirish Animatsiyalar va harakatlar qo‘shish Masalan, HTML sizga bir quti beradi. CSS esa bu qutini qizil rangda, katta va yumaloq qilib, ekranda chiroyli ko‘rinadigan qiladi. CSS – bu saytni chiroyli va jozibali qilish tili, u sizning HTML skeletingizni bezaydi va go‘zal ko‘rinishini ta’minlaydi.

JavaScript

JavaScript – bu saytga harakat va interaktivlik beradigan til. Agar HTML saytning skeleti, CSS esa kiyimi bo‘lsa, JavaScript – bu saytning jonlantiruvchi kuchi. JS yordamida siz quyidagilarni qilishingiz mumkin: Tugmani bosganda nimadir sodir bo‘lishi Raqamlar, matn yoki rasm o‘zgarishi Animatsiyalar va harakatlar qo‘shish Foydalanuvchi bilan suhbat yoki o‘yinning ishlashi Masalan, siz tugmani bosganingizda rasm o‘zgaradi yoki matn harakat qiladi – bularning hammasini JavaScript qiladi. Shunday qilib, JavaScript – saytni jonli va interaktiv qilish tili. U HTML va CSS bilan birga ishlaganda sayt nafaqat chiroyli, balki ishlaydigan va qiziqarli bo‘ladi.
